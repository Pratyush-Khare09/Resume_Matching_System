## Description
This project uses Natural Language Processing (NLP) to measure similarity between a resume and job description.

## Features
- Text preprocessing
- TF-IDF vectorization
- Cosine similarity score
- Skill matching
- Missing skill detection

## Technologies Used
- Python
- Scikit-learn
- NLP
- TF-IDF
- Cosine Similarity

## How to Run

Step 1:
Install requirements

pip install -r requirements.txt

Step 2:
Run main file

python main.py
